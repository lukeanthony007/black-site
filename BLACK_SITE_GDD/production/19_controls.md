# Control Reference

Input mappings for all actions.

---

## 32. Control Reference

### Movement

| Key | Action |
|-----|--------|
| W/A/S/D | Move |
| Mouse | Look |
| Space | Jump |
| Shift | Sprint |
| Ctrl | Crouch/sneak |
| C | Slide |

### Combat

| Key | Action |
|-----|--------|
| Left Click | Right hand action |
| Right Click | Left hand action |
| Mouse4/Q (hold) | Melee modifier |
| R | Reload |
| G | Drop weapon |
| E | Interact/pickup |
| 1/2 | Select hand for swap |

### Takedown (Automatic)

Click with weapon → If in range + cursor on target + behind + unaware → Takedown
