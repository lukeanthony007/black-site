# Gadget System

Dishonored-style gadgets replacing magic.

---

## 9. Gadget System (Dishonored-Style)

### Design Philosophy

Gadgets replace magic. They should enable **creative problem-solving** and **multiple approaches**, not just "better weapons."

### Gadget Categories

#### Traversal Gadgets
| Gadget | Function | Dishonored Equivalent |
|--------|----------|----------------------|
| Grapple launcher | Quick vertical movement | Blink |
| Magnetic gloves | Climb metal surfaces | Wall climb |
| Fold-out glider | Controlled descent | Wind blast |
| Zipline gun | Horizontal traversal | Far reach |

#### Stealth Gadgets
| Gadget | Function | Dishonored Equivalent |
|--------|----------|----------------------|
| Sound decoy | Lure guards | Possession distraction |
| Holographic projector | Create fake person | Doppelganger |
| Smoke capsule | Visual cover | Void gaze |
| EMP burst | Disable electronics | Windblast (disabling) |

#### Observation Gadgets
| Gadget | Function | Dishonored Equivalent |
|--------|----------|----------------------|
| Heartbeat sensor | Detect nearby NPCs | Dark vision |
| Camera tap | View security feeds remotely | Void gaze |
| Audio bug | Overhear conversations | Eavesdrop |
| X-ray monocle | See through walls (limited) | Dark vision |

#### Manipulation Gadgets
| Gadget | Function | Dishonored Equivalent |
|--------|----------|----------------------|
| Remote hacking tool | Door/camera control at range | Possession |
| Stun dart | Non-lethal takedown | Sleep dart |
| Rewire tool | Trap modification | Rewire |
| Noisemaker mine | Triggered distraction | Springrazor (non-lethal) |

#### Combat Gadgets
| Gadget | Function | Dishonored Equivalent |
|--------|----------|----------------------|
| Flashbang | Blind and stun | Windblast |
| Sticky shocker | Electric incapacitation | Arc mine |
| Gas grenade | Area denial | Plague rats |
| Deployable turret | Automated defense | Wall of light |

### Gadget Economy

- **Gadgets cost budget** (pre-mission purchase)
- **Limited charges/uses** per mission
- **Resupply possible** (find in facility, buy from insider)
- **Role restrictions** (some gadgets require training/role)

### Creative Combinations

**The goal:** Players discover emergent solutions.

| Combination | Effect |
|-------------|--------|
| Smoke + fire alarm | Evacuation without suspicion |
| EMP + backup generator | Selective blackout |
| Sound decoy + tripwire | Trap guards in loop |
| Camera tap + remote hack | Open door when guard looks away |
| Hologram + real disguise | Confuse patrol patterns |
